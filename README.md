# Interactive Date Invitation App

An elegant, step-by-step interactive invitation card web application built with vanilla HTML, CSS, and JavaScript. The app includes an opening envelope mechanism, custom step-by-step interactive choices for dates (utilizing a dynamic calendar interface), locations, and times, culminating in an email RSVP dispatch confirmation.

## Features
- **Interactive Wax-Seal Envelope:** A CSS-animated envelope transition that sets a classic tone.
- **Dynamic Grid Calendar:** Fully operational date constraints preventing past selection.
- **Visual Option Selection:** Clean, responsive grid lists for choosing venues and time frames.
- **Automatic RSVP Email Trigger:** Compiles user choices dynamically and populates an automated email mailto link format.
